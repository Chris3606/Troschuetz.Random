TRandom.

BinomialSamples(double, int) Method

Summary

Returns an infinite sequence of binomial distributed 32-bit signed integers.
Namespace
Troschuetz.Random
Containing Type
TRandom

Syntax

public IEnumerable<int> BinomialSamples(double alpha, int beta)

Remarks

This method simply wraps a call to Next(), with a prior check of the distribution parameters. If you absolutely need the best performance, you may consider using an instance of BinomialDistribution for each group of parameters.

Parameters

Name Type Description
alpha double The parameter alpha which is used for generation of binomial distributed random numbers.
